## 🎯 Key Technical Achievements

### 1. **Robust DOM Selectors**
Uses ARIA labels and data attributes instead of brittle CSS classes:
```typescript
// Stable selector priorities:
[role="button"][data-item-id]  // Primary
[aria-label*="..."]             // Fallback
CSS class selectors             // Last resort
```

### 2. **Email Extraction**
- Strict RFC-5322 compliant regex
- Smart email selection (contact@ > info@ > support@)
- Filters false positives (noreply@, bounce@, test@, etc.)

### 3. **Rate Limiting**
- 2.5-second delay between web fetches
- Request queue prevents server blocking
- Graceful error handling for CORS/network issues

### 4. **Type-Safe Data Flow**
- TypeScript interfaces ensure consistency across components
- Chrome storage manager with explicit typing
- Compile-time error detection

### 5. **User Experience**
- Real-time dashboard updates every 1 second
- Status messages for all user actions
- Responsive dark/light mode design
- Accessible checkbox controls

## 🐛 Troubleshooting

### "Could not find Google Maps results list"
- Ensure you're on a Google Maps search page with results
- Try a new search
- Google Maps DOM may have changed - check selectors in `content.ts`

### Extension not appearing in Chrome
- Ensure Developer Mode is enabled (`chrome://extensions/`)
- Hard refresh the extensions page
- Try "reload" button next to extension name

### No emails being found
- Many sites don't store emails in HTML source (JavaScript-rendered)
- Try specific sites known to have contact pages
- Current implementation limited to homepage + /contact + /about

### Service Worker not responding
- Go to `chrome://extensions/` → TeraLead → "Service Worker"
- Check console for errors
- Try reloading the extension

## 💡 Architecture Decisions

### Why Service Worker for Email Fetching?
Content scripts have CORS restrictions. Service workers can fetch from any domain, making them ideal for website scraping.

### Why Smart Scrolling?
Google Maps lazy-loads results. Smart scrolling with delays ensures all content loads before extraction.

### Why Type-Only Imports?
Reduces bundle size and prevents circular dependencies in extension environment.

### Why ARIA Labels?
Google Maps changes CSS classes frequently. ARIA labels are more stable and represent actual content semantics.
